Model Architecture

  • STARFlow: 3B parameters for text-to-image
  • STARFlow-V: 7B parameters for text-to-video
  • Transformer autoregressive flow architecture
  • Up to 640×480 resolution

Generation Settings

  • Resolution: 640×480 (480p)
  • Frame rate: 8-24 FPS
  • Length: 5-30+ seconds
  • Classifier-free guidance: 3.5-3.6

Key Features

  • High temporal consistency
  • Fast Jacobi iteration sampling
  • Text and image conditioning
  • Variable aspect ratios
